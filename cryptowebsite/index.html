Kryptowährungen

Was sind Kryptowährungen?

Kryptowährungen sind digitale oder virtuelle Währungen, die Kryptographie für die Sicherheit nutzen. Eine der bekanntesten Kryptowährungen ist Bitcoin.

Wie funktionieren sie?

Kryptowährungen basieren auf der Blockchain-Technologie, die eine dezentrale und sichere Speicherung von Transaktionsdaten ermöglicht. Jede Transaktion wird in einem Block gespeichert und an eine Kette von vorhergehenden Transaktionen angehängt, daher der Name "Blockchain".

Beliebte Kryptowährungen

  • Bitcoin (BTC)
  • Ethereum (ETH)
  • Ripple (XRP)
  • Litecoin (LTC)
  • Cardano (ADA)
